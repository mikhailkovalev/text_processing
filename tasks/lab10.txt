1. Описать структуру данных  для хранения  расписания занятий с указанием номера  группы, дня недели, номера    лекции, фамилии     и аудитории. Предполагается, что расписание занятий хранится в файле, имеющем структуру:
Название группы
День недели
Дисциплина: преподаватель,аудитория
Дисциплина: преподаватель,аудитория
...
Реализовать процедуру чтения данных из файла в структуру данных и наоборот. (6 баллов)

2. Описать структуру данных для хранения шахматной партии. Предполагается, что данные о партии записываются в файл, имеющий структуру:
ФИО шахматиста "белых" - ФИО шахматиста "черных"
Дата,город
1. ход белых ход черных
2. ход белых ход черных
...
Ход записывается в виде: e2-e4 или Кg8-f6. 
Реализовать процедуру чтения данных из файла в структуру данных и наоборот.
(6 баллов)

3. Описать структуру данных для хранения телепрограммы. Предполагается, что данные записаны в файле в виде:
Название телеканала
День недели - дата
чч:мм Название телепередачи
чч:мм Название телепередачи
...
Реализовать процедуру чтения данных из файла в структуру данных и наоборот.
(6 баллов)

4. Описать структуру данных для хранения расписания движения поездов. Предполагается, что данные записаны в файле в виде:
Номер поезда
Название станции время	прибытия	время отправления
Название станции время	прибытия	время отправления
...
В каждой стоке данные разделяются символом табуляции.
Реализуйте процедуры чтения данных из файла в структуру данных и наоборот.
(6 баллов)


5. Для задачи 4 реализуйте процедуру подсчета общего времени в пути.


6. Для задачи 4 реализуйте процедуру построения ассоциативного массива. В этом массиве ключи -- названия станций, а значениями являются количества минут стоянки поезда.
