
1. Запись многочлена имеет вид
1.1+2.3x+5.12x^2+11.2x^12-1.7x^32
С помощью регулярного выражения извлечь из строки хеш:
ключ - показатель степни,
значение - коэффициент

2. Состаивть регулярное выражение для записи химическй формулы вещества в виде:
H2O, C2H5OH, NaCl, H2SO4.

3. С помощью регулярного выражения вычислить значение выражения вида
11-2*31+24:4-72 извекая из такой строки массив цеых чисел и массив операций.

4. Символьная строка содержит науральные числа. Найти сумму тех из них, коорые не превышают 256 не прибегая к использованию оператора if. (Т.е. проверку выполнение неравенства осуществить с помощью регулярных выражений -- по записи числа, а не по его значению.)

5. Составить регулярное выражение для записи вида
21(x-2)(x-4)(x+21)(x+7)...(x-3).
Извлечь из такой текстовой строки массив корней этого многочлена.

6. В файле записаны почтовые адреса, разделенные пустыми строками.
Отсортировать почтовые адреса по значению почтового индекса, составив хеш:
ключ --  индекс,
значение -- массив соответствующих адресов.

7. Файл содержит список литературы в формате

[12] Препарата Ф., Шеймос М. Вычислительная геометрия: Введение. М.: Наука, 1989, 478 с.

Составить регулярное выражение ля автоматического составления хеша с ключами 'number','authors', 'name', 'publisher', 'year', 'pages'.
