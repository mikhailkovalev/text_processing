1. Из файла формата xml   с помощью регулярных выражений составить хеш массивов. Ключами хеша дожны быть названия тегов, а значениями массивы хешей. Каждый такой массив соответствует одному вхождению тега в файл, а элементами являются хеши с ключами-атрибутами и их значениями. Пример
<img src="file.png" width=200>
...
<img src="pic.jpg" height=100>

H{img}=[{src=>'file.png', width=>'200'},{src=>'pic.jpg',height='100'}] 

2. Выбрать из текста слова, содержащие только глухие согласные. 

3. Выбрать из текста слова, содержащие шипящие звуки. 

4. Найти в тексте все "симметричные"  целые числа: 11, 121, 2332, 49194 ,548845. Число знаков не превосходит 7.

5. По заданному тексту составить хеш. Ключи хеша -- строки вида "ccvc",  а значение -- массив слов текста с заданноым правилом чередования гласных (v)  и согласных (c).

6. Строка содержит запись вида "34<528". Проверить правильность неравенства по записи чисел, а не поих значениям.
