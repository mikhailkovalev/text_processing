Все задачи выполняются в виде процедур.


1.  Записать числа из  файла в двумерный массив (матрицу).При этом каждой строке в файле соответствует строка в матрице. Недостающие числа заполняются нулями справа. Вычислить массив сумм чисел столбцов этой матрицы.

2. Реализовать процедуру умножения двух мариц. Процедура получает два параметра: ссылки на матрицы. Недостающие элементы заменить нулями.

3. Задан массив хешей. Ключи хеша: id,email, name, old,  а значениями являются данные, взятые из БД. Построить массив тех хешей, значения в которых, соответствующих ключу email  начинаются с заданной буквы, а поле  id  удовлетворяет неравенству 10<id<1000.

4. Для заданного текста составить массив слов, ключами которого являются букв, а значениями -- массивы слов, начинающихся с соответствующей буквы.

5. Для заданного текста составить хеш массивов, ключами которого являются слова, а значениями массивы номеров предложений, в которых это слово встречается.

6. Задана двумерная матрица. Найти минимальный элемент среди максимальных элементов строк этой матрицы.

7. Задан массив хешей. Ключами хешей являются station, tin, tout. Соответствующие им значения:  название ж/д станции, время прибытия и время отправления в формате чч:мм.  Вычислить полное время в пути.
