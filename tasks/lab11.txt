1. Пакет Users реализуйте в модуле Users.pm. В пакете должен быть определен массив хешей -- массив записей о пользователях. Каждый элемент массива -- это хеш к ключами: id, name, email, age, status. В   модуле реализуйте процедуры:
	а) добавить информацию о новом пользоватее в массив;
	б) обновить данныео пользователе с данным значением  ключа id;
	в) сохранить данные о пользователях в файл;
	г) загрузить данные о пользоватеассив пакета.

2. Пакет Students реализуйте в модуле Students.pm. В пакете должен быть определен массив хешей -- массив записей о студентах. Каждый элемент массива -- это хеш к ключами: id, name, group, marks. Ключу  marks  должен соответствовать другой хеш с ключами -- названиями дисциплин, а значенями должны быь списки баллов студента по данной дисциплине. В   модуле реализуйте процедуры:
	а) добавить информацию о новом студенте в массив;
	б) добавить информацию о баллах заданному студенту для заданной дисциплине;
	в) сохранить данные о студенте в файл;
	г) загрузить данные о студенте в массив пакета.

3.  В модуле TextAnalisys.pm  реализуйте одноименный пакет с 
процедурами выполняющие действия:
	а) подсчет числа символов в тексте,
	б) подсчет числа слов в тексте,
	в) вычисление хеша со значениями частот символов текста,
	г) вычисление хеша со значениями частот слов текста,
	д) сортировка соответствующих хешей по значению частот.
	
4. Создайте модуль Trianngle.pm  с реализацией одноименного пакета. 
В пакете реализуйте процедуры хранения данных о треугольнике и
вычисления характеристик треугольника:
	а) периметр треугольника,
	б) площадь треугольника,
	в) массив углов треугольника,
	г) радиус описанной окружности,
	д) радиус вписанной окружности.
