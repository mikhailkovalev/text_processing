1. В файле в каждой строке записано слово. Перед ударной гласной стоит знак !.  Для заданного текста определить последовательность ударных и безударных гласных по словарю из файла. 

2. Выбрать из текста слова, содержащие только глухие согласные. 

3. Выбрать из текста слова, содержащие шипящие звуки. 

4. Выбрать слова из текста, содержащие только звонкие согласные. 

5. Для двух заданных слов средствами регулярных выражений найти максимальное совпадение окончания первого слова и начало второго.
Например, для слов "пруд"  и "руда" мы получаем "руд", для слов  "удача" и "часы" мы получаем  "ча".
