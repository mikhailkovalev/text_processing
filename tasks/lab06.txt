

1. Построить структуру данных для хранения слов текста в виде двумерного массива. Индексы: номер предложения и номер слова в предложении. Реализуйте процедуры добавления нового предложения текста в структуру, а также подсчет частоты слов во всем тексте и в каждом предложении отдельно.

2. Для заданого текста построить хеш массивов для расписания занятий:
 ключ -- день недели, значение  -- массив дисциплин. Реализовать процедуры:
 добавления дисциплины в данный день недели, изменения дисциплины, вывод списка дисциплин в заданный день недели и вывод всего расписания.

3. Для заданного текста построить хеш массивов: ключ -- слово текста, значение -- массив номеров предложений, в которые оно входит.

4. Составить хеш: ключ -- слово, значение сколько раз слово встретилось в заданном тексте.
